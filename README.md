# 연습내용
### django
- 다양한 모델 관계 이용 ( 1:1, 1:N, M:N )
- 쿼리셋 최적화( Django debug toolbar 활용 )
- dropzone 라이브러리 이용해서 이미지 업로드
- 태깅 기능 구현

### 배포
- 파이썬 애니웨어를 활용한 배포 연습
- PostgreSQL 데이터베이스 연동
- 실행 환경에 따른 (개발/배포) requirements.txt, settings.py 파일 분리
